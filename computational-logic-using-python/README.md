# 💻 Computational Logic Using Python

## 📖 Descrição
Este projeto reúne exercícios e práticas desenvolvidas durante o estudo de lógica computacional utilizando a linguagem Python.

O objetivo é demonstrar a aplicação dos conceitos fundamentais da programação através de exemplos práticos.

---

## 🎯 Objetivo
Desenvolver o raciocínio lógico e a capacidade de resolução de problemas utilizando Python, abordando:

- Estruturas condicionais
- Estruturas de repetição
- Manipulação de dados
- Organização de código

---

## ⚙️ Conteúdos abordados

- 📥 Entrada e saída de dados
- 🔀 Estruturas condicionais
- 🔁 Estruturas de repetição
- 🧩 Funções e organização lógica
- 🛠️ Operações e manipulação de dados

---

## 🛠️ Tecnologias utilizadas

- Python
- Lógica de programação
