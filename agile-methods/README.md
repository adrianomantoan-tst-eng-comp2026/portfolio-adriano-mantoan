## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da disciplina de Pensamento Algorítmico e Introdução à Programação Orientada a Objetos – UniFECAF.

O objetivo foi simular a gestão de um projeto real de desenvolvimento de software utilizando metodologias ágeis, com foco em:

- Scrum
- Kanban
- Trello

---

## 🏢 Contexto do Projeto

O cenário simula uma startup de desenvolvimento de aplicativos móveis, com uma equipe composta por:

- Rute
- Adilson
- Paulo
- Geraldo
- Júnior

👨‍💼 Product Owner: Adriano Mantoan

🎯 Objetivo:
Criar uma nova versão de um aplicativo com melhorias de interface, correções de bugs e novas funcionalidades.

---

## 🔄 Metodologias Utilizadas

### 🟢 Scrum
- Sprint Planning
- Daily Scrum
- Sprint Review
- Sprint Retrospective

### 🔵 Kanban
- Fluxo visual de tarefas
- Controle de progresso
- Gestão de gargalos

---

## 📊 Estrutura do Trello

O projeto utilizou as seguintes listas:

- Product Backlog
- Sprint Backlog
- Em Andamento
- Em Teste
- Concluído
- Encerramento

---

## ✅ Benefícios Obtidos

- Melhor organização do projeto
- Transparência das atividades
- Comunicação eficiente da equipe
- Entregas contínuas

---

## ⚠️ Desafios Encontrados

- Manter o quadro atualizado
- Priorização correta das tarefas
- Controle de qualidade vs velocidade

---

## 🧠 Decisões Estratégicas

- Priorização de bugs críticos
- Apoio à equipe em impedimentos
- Acompanhamento contínuo das sprints
