# 🖥️ Computer Architecture and Organization

## 📖 Descrição
Este projeto apresenta a simulação e análise da arquitetura de um computador, com foco nos principais componentes: CPU, memória, dispositivos de entrada/saída e interconexões.

O trabalho foi desenvolvido com base em conceitos fundamentais da arquitetura de computadores, permitindo compreender como os dados são processados internamente em um sistema computacional.

---

## 🎯 Objetivo
Modelar uma arquitetura computacional completa, demonstrando:

- Funcionamento da CPU
- Organização da memória
- Comunicação entre componentes
- Execução de instruções (ciclo de máquina)

---

## ⚙️ Conteúdos abordados

- 🧠 Arquitetura de Von Neumann
- ⚙️ CPU (Unidade de Controle e ALU)
- 🧮 Registradores
- 💾 Memória (RAM, ROM e Cache)
- 🔁 Ciclo de instrução (Fetch → Decode → Execute)
- 🔌 Barramentos (dados, controle e endereço)
- ⌨️ Dispositivos de entrada e saída

---

## 🛠️ Tecnologias e ferramentas utilizadas

- Logisim (simulação lógica)
- Tinkercad Circuits
- Draw.io (diagramas)
- Conceitos teóricos de arquitetura computacional

## 📊 Entregáveis do projeto

O projeto foi dividido em três partes principais:

### 📘 Parte Teórica
- Explicação da arquitetura utilizada (ex: Von Neumann)
- Conceitos de CPU, memória e registradores
- Descrição do ciclo de execução de instruções

### 🛠️ Parte Prática
- Criação de diagrama funcional da arquitetura
- Representação dos componentes do sistema

### 🎥 Apresentação (Vídeo Pitch)
- Explicação do funcionamento da arquitetura
- Demonstração do fluxo interno do computador
- Comunicação técnica de forma didática
