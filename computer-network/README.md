# 🌐 Computer Network Project (Rede para Loja de Varejo com CFTV)

## 📖 Descrição
Este projeto consiste no planejamento e desenvolvimento de uma rede de computadores para uma loja de varejo, integrando sistema de controle de estoque, dispositivos de pagamento e monitoramento por câmeras de segurança (CFTV).

A solução foi projetada para garantir comunicação eficiente, segurança dos dados e funcionamento simultâneo de todos os dispositivos.

---

## 🎯 Objetivo
Desenvolver uma arquitetura de rede funcional que atenda a um cenário real de negócio, garantindo:

- Conectividade estável entre dispositivos
- Acesso simultâneo ao sistema de estoque
- Monitoramento contínuo por CFTV
- Processamento seguro de pagamentos
- Armazenamento confiável de dados

---

## ⚙️ Cenário do projeto

O ambiente simulado representa uma loja de varejo contendo:

- 🧾 3 caixas registradoras
- 📱 2 tablets
- 🖨️ 1 impressora
- 🖥️ 1 servidor local (estoque e CFTV)
- 🎥 4 câmeras IP
- 💳 1 máquina de cartão

---

## 🧠 Conceitos aplicados

- LAN (rede local)
- WAN (internet)
- WLAN (rede sem fio)
- Endereçamento IP
- DHCP
- DNS
- Gateway

---
