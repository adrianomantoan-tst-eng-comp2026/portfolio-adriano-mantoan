## 📌 Sobre o Projeto

Este projeto demonstra a criação e configuração de um **Servidor Web utilizando Windows Server 2022 com IIS**, em ambiente virtualizado com **VirtualBox**, simulando um cenário real de hospedagem.

---

## 🎯 Objetivos

* Instalar e configurar o IIS
* Criar e publicar um site local
* Entender o funcionamento do HTTP e HTTPS
* Aplicar segurança com certificado digital
* Simular um ambiente real de servidor

---

## 🛠️ Tecnologias Utilizadas

* Windows Server 2022
* IIS (Internet Information Services)
* VirtualBox
* HTTP / HTTPS
* Certificado SSL (Self-Signed)

---

## ⚙️ Etapas do Projeto

### 🔹 1. Criação da Máquina Virtual

* Instalação do Windows Server 2022 no VirtualBox

### 🔹 2. Instalação do IIS

* Ativação do recurso IIS no sistema

### 🔹 3. Configuração do Servidor

* Criação do site
* Definição de diretório
* Configuração de porta

### 🔹 4. Publicação do Site

* Teste de acesso via navegador

### 🔹 5. Implementação do HTTPS

* Criação de certificado digital
* Ativação da porta 443
* Acesso seguro via HTTPS

---

## 🔐 HTTP vs HTTPS

| HTTP             | HTTPS         |
| ---------------- | ------------- |
| Sem segurança    | Seguro        |
| Sem criptografia | Criptografado |
| Porta 80         | Porta 443     |
